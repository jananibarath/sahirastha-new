# Content source index for SahiRastha

This folder is the implementation reference set for Codex.

## How to use this folder
- Start with `current/`.
- Use `spreadsheets/` for service-batch context and cross-checking.
- Use `archive/` only if a current file is silent or if older drafting context is needed.

## Priority and precedence
1. Tightened / cleaned / current documents override older drafts.
2. Structure files define page architecture.
3. Copy files provide implementation copy and CTA language.
4. If a file explicitly says that numbers are placeholders, do not publish those numbers until verified.
5. Draft testimonial and case-story models are not approved testimonials.
